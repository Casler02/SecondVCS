VCS project / Made by Wyatt Casler / May 2019

Why is this so important?

A VCS such as Github is mainly used as a way for people to work together on a project in order to revise things at the same time as others who might be working on a different part of the project.

Step by step process

compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git status
    
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git commit -m "First ever commit"
    
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git log
    
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs/GitHubClones$ git clone https://github.com/CHS-McClendon/VCS_mrm.git .
    
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git remote add origin https://github.com/Casler02/SecondVCS.git
    
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git pull origin master --allow-unrelated-histories
ALWAYS DO THIS BEFORE A PUSH! 
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git push origin master
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git branch testbranch
    
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git branch
    
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git checkout testbranch
    
compsci@compsci-VBox:/var/www/html/wyattc/projects/vcs$ git push -u origin testbranch
